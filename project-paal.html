// Overview

The Problem & Solution

USF students waste hours in office-hour queues or struggling to phrase questions for generic AI chatbots that lack course context. PAAL solves this with a study assistant that understands USF's curriculum natively: students ask naturally and get instant, contextual help. No prompt engineering required.

Built entirely through vibe coding: AI-assisted development from concept to production.

// Key features

What Makes PAAL Different

Promptless Interaction
Students ask naturally, like "explain polymorphism in Java", and get course-specific, step-by-step responses. Zero prompt engineering needed.
Smart Quiz Engine
AI-generated quizzes adapt to each course. Choose topic, difficulty, and question count. Instant feedback with detailed explanations.
Course-Aware Chat
Chat maintains course context. Select enrolled courses and the AI adapts its knowledge base for each subject.
Full Course Catalog
Searchable catalog of 16+ USF courses across 8 departments. Filter by department, search by code or name.
Progress Tracking
Analytics dashboard visualizes learning progress, identifies weak areas, and generates personalized study recommendations.
Gamification
Badges, streaks, and milestones keep students motivated through study sessions and concept mastery.
