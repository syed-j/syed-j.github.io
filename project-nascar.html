ETL Architecture

Race data was fragmented across 20 individual files: each race had a separate pit stop CSV and results XLSX with inconsistent column names, mixed date formats, and no linking keys. Built a repeatable Tableau Prep flow that unions all sources, standardizes schemas, handles outliers, and engineers new calculated fields like Total Jack Time.

Insights & Outcomes

18.4s Crew Performance Gap
The fastest crew (Car #12 at 38.4s) outperformed the slowest (Car #43 at 56.8s) by 18.4 seconds per stop, potentially costing 2+ positions per pit cycle.
Track Type Drives Pit Time
Road courses (Watkins Glen, 55.3s) averaged 27% slower pit times than ovals (Homestead, 43.6s), reflecting different pit lane configurations.
20 to 2 File Consolidation
Automated Tableau Prep flow eliminated manual copy-paste workflows, engineered composite join keys (Race # + Car #), and created calculated fields for deeper analysis.
Manufacturer Parity
Wins split nearly evenly: Chevrolet (4), Toyota (3), Ford (3), despite Ford and Chevy having 2.7x more entries than Toyota.
